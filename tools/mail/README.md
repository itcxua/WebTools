# MailCheck — PowerShell Mail Audit Tool

### 📋 Возможности
- Проверка Reverse DNS (PTR)
- Проверка MX, SPF, DKIM, DMARC
- Проверка TLS (STARTTLS)
- Проверка репутации IP и домена (Spamhaus, MXToolbox)
- Меню-интерфейс, запуск из CMD
